# Learner prefers breadth-first across the funnel, not depth-first on one skill

Offered the choice to drill on hooks (drafting from a real sermon) before advancing,
the learner instead chose to build Lesson 0002 (Retain) immediately. Signal: they want
to see the whole strategic arc laid out before grinding any single skill to mastery.

**Implication for future sessions:** keep building the funnel lessons in order
(0003 Minister next) so the full map exists, but keep each lesson's hands-on *task*
open and revisitable. Don't gate the next lesson on completing the prior task — the
learner pulls the map together first, then practices. Expect them to return for
feedback on real hooks/CTAs once the arc is visible. Pacing = breadth now, depth on demand.
